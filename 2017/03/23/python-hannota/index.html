python实现汉诺塔

本文是学习廖雪峰的python3.0写的关于递归的作业，含有思路，希望对同学们有帮助。

1.建立递推规则

思路：将第n个盘子从a放到c可以分三步走：
1.将n-1个盘子从a经过c放到b
2.将a中剩下的第n个盘子放到c
3.将n-1个盘子从b经过a放到b
即：递推规则为
move（n,a,b,c）= move（n,a,c,b）、a-》c、move（n,b,a,c）

2.设定递归终结条件

思路：考虑起始塔剩下2个盘如何移动到目标塔的方法，也分三步
1.起始塔-》目标塔
2.起始塔-》第三方塔
3.第三方塔-》目标塔

3.小技巧

将所有参数封装到一个长度为4的list中，设为paras，paras[0]存放盘数，paras[1]代表起始塔，paras[2]代表第三方塔，paras[3]代表目标塔。注意递归调用时，参数list的调整。
